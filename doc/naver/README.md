## 네이버 로그인 활용 관련

네이버 아이디 로그인을 통해 제공 받는 모든 항목

- `id`
- `email`
- name 관련 : `displayName`, `nickname`

```json
{
  "provider": "naver",
  "id": "34057140",
  "displayName": undefined,
  "emails": [{ "value": "tkhwang@gmail.com" }],
  "_json": {
    "email": "tkhwang@gmail.com",
    "nickname": undefined,
    "profile_image": "https://ssl.pstatic.net/static/pwe/address/img_profile.png",
    "age": undefined,
    "birthday": undefined,
    "id": "34057140"
  }
}
```
